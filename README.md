    # Фиксация времени завершения выполнения программы

 ```python
import math


def square(a, b, c):
    p = (a + b + c) / 2
    if (p-a <= 0) or (p-b <= 0) or (p-c <= 0):
        return "Такого треугольника не существует!"
    s = math.sqrt(p * (p - a) * (p - b) * (p - c))
    return s


```

# Общий вывод 


В Python функции создаются с помощью ключевого слова def, за которым указывается имя функции и список параметров, заключенных в круглые скобки. Функции могут возвращать значения с помощью ключевого слова return. 
Переменные, объявленные внутри функции, называются локальными и доступны только в пределах этой функции. В то время как переменные, созданные вне функций, являются глобальными и могут быть использованы в любой части программы.
